 # My Neovim Configuration  
- Leader = space
## main-keymap 

### Terminal
|keybinding| description|
|--------|----------------|
| leader-tt | open terminal in current file directory |
| leader-tr | open terminal in root |

### Function
|keybinding|description|
|----------|--------------|
| leader-md | preview markdown with glow |
| leader-e | open nertw |

## window navigatoin 
|mode|keybinding|action|
|-----|-----|---------------|
|Normal|leader-hjkl | switch windows|
|Terminal|leader-hjkl | switch windows|

## requirements
- Neovim >= 0.10
- Git
- 'glow'(for markdown preview)
